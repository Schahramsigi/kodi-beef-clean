# kodi-beef-clean

A cleanup utility for Kodi Artwork Beef.

## Features

- Removes duplicate artwork
- Prefers Artwork Beef filenames
- Dry-run mode
- Delete mode
- Interactive mode
- Statistics
- Trash support

## Supported artwork

- Poster
- Fanart
- ClearLogo
- ClearArt
- Discart
- Banner
- Landscape
- Thumb

## Requirements

- Kodi 21+
- Artwork Beef
- Bash 5.x
